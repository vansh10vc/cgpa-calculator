# Grade Ledger — SGPA & CGPA Calculator

A weighted-average GPA calculator for the 10-point Indian grading scale.
Computes SGPA from a single semester or CGPA when you add prior standing.

**Live:** https://yourusername.github.io/cgpa-calculator

## Built with
- Vanilla HTML, CSS, JavaScript
- No frameworks, no dependencies
- Fraunces + Instrument Serif + JetBrains Mono via Google Fonts

## Formula
Weighted average: Σ(credit × grade) / Σ(credit)
